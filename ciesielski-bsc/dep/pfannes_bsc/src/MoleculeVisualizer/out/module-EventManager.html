Module: EventManager

Responsible for managing all the events inside the Molecule Visualizer. It is able to add multiple event listeners to an object. The event functions for each listener have to be provided as well. Objects are required to derive from the EventTarget interface so the EventManager can add and remove event listeners. It keeps a JSON-like map containing all objects with their respective events and event functions.
