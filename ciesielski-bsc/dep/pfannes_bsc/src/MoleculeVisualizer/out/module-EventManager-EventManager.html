Methods

addEvents(object, events, eventFunctions)

Takes an object object and a number of events and adds listeners for each of those events to object. The functions for each listener are specified in eventFunctions.
Parameters:
Name Type Description
object object Object for which to add the event listener for. Needs to extend the EventTarget interface so event listeners can be added to the object. It is possible to add multiple listeners to a single object by specifying more than one event in events. events and eventFunctions should have the same length as it is assumed that the i-th function corresponds to the i-th event. After adding the listeners, the object is stored inside the managedEvents object together with the listened events and the corresponding event functions.
events Array.<string> Events which the object should listen for.
eventFunctions Array.<function()> Event functions that should be executed if their corresponding event has been triggered.

// this.managedEvents = {object: {event: eventFunction}} after call.
