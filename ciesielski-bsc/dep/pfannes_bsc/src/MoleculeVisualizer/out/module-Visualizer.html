Module: Visualizer

Main module for the Molecule Visualizer. Enables parsing of molecule data in .pdb format and visualization of said molecules. At the moment there are 3 different models you can choose from: Balls and Sticks model, Wireframe model and Van der Waals model. Visualization of surface models using the .stl file format is also possible. It's also possible to view molecules via VR goggles. Includes a small GUI for toggling which model should be used during visualization, which molecule to display (currently a dropdown list with 2 test molecules) and an FPS counter to keep track of the performance of the visualization.
